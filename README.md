# Rosetta

Unofficial implementation of Rosetta range filter, published in [SIGMOD2020](https://stratos.seas.harvard.edu/files/stratos/files/rosetta.pdf).

## Build and Run

`rose_test.c` provides a simple demo on how to use the filter.

```
mkdir build
cd build
cmake ..
make -j8
```
